# 遊戲設置

## 官方設置整理

| 步驟 | 規則 |
|---|---|
| 板圖 | 放在桌面中央。 |
| 目的地 | 貿易點與定居點各自洗牌，分別翻開 3 張。 |
| 船員牌庫 | 2 人移除左下角標有 3 或 4 的船員；3 人移除標有 4 的船員；4 人使用全部。 |
| 船員市場 | 洗混剩餘船員後翻開前 5 張。 |
| 玩家配件 | 每人拿 1 塊船隻板塊、3 枚銀臂鐲、3 枚招募標記。 |
| 船隻面 | 全體玩家共同決定基礎面或進階面；首次建議基礎面。 |
| 起始船員區 | 每人從牌庫拿取 2 張不同顏色船員，正面朝上放在自己的船員區。 |
| 起始標記 | 每人將 1 枚招募標記與 1 枚銀臂鐲放入船隻兩側指定位置。 |
| 分數/聲望 | 圓形木標記放計分軌起始格；方形木標記放聲望軌起始格。 |
| 起始手牌 | 每人抽 3 張船員。 |
| 起始玩家 | 從本局船隻顏色對應的命運標記中隨機抽 1 枚決定。 |

## 2 人玩家 vs 電腦 preset

實作資料位於 `data/setup_presets.json`：

- 玩家：`human` 與 `computer`。
- 船員牌庫：移除人數標記 3、4。
- 公開目的地：貿易點 3 張、定居點 3 張。
- 公開船員：5 張。
- 起始船員區：2 張不同色船員。
- 起始手牌：3 張。
- 起始標記：招募 1、銀臂鐲 1。
- 起始玩家：命運標記隨機抽選。
- 文字模擬船隻面：`human` 使用 `knarr_core_ship_tile_1_side_001`，`computer` 使用 `knarr_core_ship_tile_2_side_003`。
